# 🏏 Chennai Super Kings Fan Page 💛

A visually rich and modern **CSK Fan Website** built using pure HTML, CSS, and JavaScript.
Designed to capture the spirit of the **Yellow Army** with smooth animations, interactive UI, and IPL-style aesthetics.

---

## 🔥 Live Demo

👉 https://challa-koushik.github.io/ShadowFox/csk/

---

## ✨ Features

* 🎨 **Premium UI Design** inspired by IPL theme
* 🎥 **Animated Hero Section** with dynamic background
* 🏆 **Trophy Cabinet** showcasing CSK achievements
* 👥 **Squad Section** with player cards
* 📅 **Match Schedule UI** (IPL-style fixtures)
* 🌟 **Legends Section** highlighting iconic players
* 💛 **Fan Zone** with chants and interactive elements
* 📱 **Fully Responsive Design** (mobile + desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Animations, Gradients, Glassmorphism)
* JavaScript (DOM + Interactivity)

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/challa-koushik/ShadowFox.git
```

2. Navigate to CSK folder:

```
cd ShadowFox/csk
```

3. Open:

```
index.html
```

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

* Go to repo → Settings → Pages
* Select `main` branch
* Root folder `/`
